# LostArk Marketplace API Service  
로스트아크 오픈 API 연동 이커머스 서비스  


## 📌 프로젝트 개요

이 프로젝트는 **로스트아크 오픈 API**와 연동된 이커머스 API 서비스를 제공합니다.  
유저는 자신의 캐릭터 정보를 연동해 게임 내 재화로 아이템을 구매하거나, 특정 랭킹 및 업적 달성 시 보너스 혜택를 지급받을 수 있습니다.  
이를 통해 **로스트아크 게임 플레이와 더불어 상점 이용 경험을 통합**합니다.


## 🛠 기능 설명  

### 1️⃣ 회원정보  
**설명**  
- 유저는 로스트아크 계정을 연동하여 캐릭터 정보를 관리합니다.

**세부 기능**  
- **프로필 조회 및 수정**: 이름, 이메일, 비밀번호 조회 및 변경.  
- **로스트아크 캐릭터 연동**: API를 사용해 게임 유저의 캐릭터 목록 및 능력치 불러오기.  
- **구매 및 결제 내역 조회**: 구매한 아이템과 결제 금액, 결제 날짜 표시.  
- **비밀번호 초기화**: 이메일 인증 후 랜덤 비밀번호 발송.  
- **게임 랭킹 연동**: 유저의 PvP 및 서버 랭킹 표시.

### 2️⃣ 상품명 검색 기능  
**설명**  
- 상점에서 구매하려는 상품을 검색할 수 있습니다.

**세부 기능**  
- **상품명 자동 완성**: 검색창에 입력 시 관련 상품명 자동 완성 제공.  
- **필터 기능**: 가격, 재화 종류, 레벨 제한 조건으로 검색 필터링.  
- **정렬 옵션**: 인기순, 가격순, 최신순 정렬 제공.  

### 3️⃣ 인게임 재화 및 포인트로 구매 기능  
**설명**  
- 게임에서 획득한 인게임 재화(골드)와 포인트를 활용해 이커머스 상점에서 아이템을 구매합니다.

**세부 기능**  
- **골드 결제**: 로스트아크 게임 재화 및 포인트를 사용해 결제.  
- **잔액 확인 및 경고**: 결제 시 부족한 경우 메시지 제공.  
- **구매 확정 시 인벤토리 반영**: 아이템 구매 후 API를 통해 유저 인벤토리에 추가.  
- **환불 기능**: 구매 취소 시 재화 복원 가능.  
- **결제 시 포인트 적립**: 결제 완료 시 추가 포인트 제공.  

### 4️⃣ 장바구니 기능  
**설명**  
- 유저가 상품을 장바구니에 담아 관리하고, 나중에 결제할 수 있습니다.

**세부 기능**  
- **상품 담기**: 동일 상품 선택 시 수량 자동 증가.  
- **장바구니 조회**: 총 가격과 결제에 필요한 골드 표시.  
- **수량 변경 및 삭제**: 상품의 수량을 조정하거나 삭제 가능.  
- **장바구니 결제**: 결제 성공 시 상품이 인벤토리로 이동.  

### 5️⃣ 포인트 적립 기능  
**설명**  
- 유저가 결제한 금액에 따라 포인트를 적립해 추가 혜택을 제공합니다.

**세부 기능**  
- **일반 포인트 적립**: 결제 금액의 3~5%를 포인트로 적립.  
- **VIP 등급 차등 적립**: 특정 캐릭터 레벨 이상은 최대 10%까지 적립.  
- **포인트 유효 기간**: 6개월 내 사용 제한.  

### 6️⃣ 레벨별 접근 제한 상품 제공  
**설명**  
- 캐릭터 레벨에 따라 상점에서 구매 가능한 상품을 제어합니다.

**세부 기능**  
- **레벨별 한정 상품**: 특정 레벨 이상의 캐릭터만 구매 가능.  
- **레벨 업 보상**: 일정 레벨 달성 시 할인 쿠폰 자동 지급.  
- **업적 달성 보상 연동**: 특정 업적 달성 시 한정 아이템 제공.  


## 🧑‍💻 기술 스택  
- **Language**: Java 17  
- **Framework**: Spring Boot 3.3.4  
- **Build Tool**: Gradle  
- **Database**:  
  - **Production**: MariaDB  
  - **Development**: H2 Database  
- **ORM**: JPA (Java Persistence API)  
- **Library**: Lombok  
- **Security**: Spring Security, JWT (JSON Web Token)  
- **Cache**: Redis (장바구니 세션 유지)  
- **Server**: Embedded Tomcat  
- **Open API**: LostArk Open API (캐릭터, 거래소 정보 연동)  
- **IDE**: Eclipse  


## 📂 프로젝트 구조  
1. **회원 정보 관리**: 유저의 로스트아크 계정과 프로필 관리  
2. **인게임 재화 및 포인트로 구매 기능**: 재화 결제 및 인벤토리 반영  
3. **장바구니 기능**: 상품 추가, 조회 및 결제 처리  
4. **포인트 적립 및 보상**: 결제 금액에 따른 포인트 제공  
5. **레벨 제한 상품 관리**: 특정 레벨 이상 접근 가능한 상품 제공  
6. **상품 검색 기능**: 필터와 정렬을 통한 상품 탐색
